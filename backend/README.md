# AgriSmart AI 🌾

**AI-Powered Pocket Kisan Advisor** for Indian Farmers

### Features
- Crop disease detection from leaf photo
- Voice input support (Hindi)
- Personalized fertilizer & remedy suggestions
- Made for Uttar Pradesh farmers

### Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: FastAPI (Python)
- **AI**: Groq LLM

### How to Run

#### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
